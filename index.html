🌐 Publicar en crubiosec.dev — Guía paso a paso
OPCIÓN A · GitHub Pages (gratis, recomendado)
  1. Crea cuenta en github.com si no tienes
  2. Crea repositorio público: crubiosec (o cualquier nombre)
  3. Sube este archivo HTML renombrado como index.html
  4. Ve a Settings → Pages → Source: Deploy from branch → main
  5. Tu URL temporal será: tuusuario.github.io/crubiosec
  6. En Namecheap → Domain List → crubiosec.dev → Manage
  7. Advanced DNS → Add record: CNAME @ → tuusuario.github.io
  8. En GitHub Pages → Custom domain → escribe crubiosec.dev
  9. Activa Enforce HTTPS ✓
  10. Espera 10-30 min. propagación DNS ✓
OPCIÓN B · Netlify (drag & drop, 2 minutos)
  1. Ve a netlify.com → Crear cuenta gratis
  2. Arrastra el index.html al área de deploy
  3. Te asigna URL automática tipo random.netlify.app
  4. Site settings → Domain management → Add custom domain
  5. Escribe crubiosec.dev
  6. Netlify te muestra los DNS records a copiar en Namecheap
  7. Namecheap → Advanced DNS → pegar los registros de Netlify
  8. Espera 15-60 min → HTTPS automático ✓
